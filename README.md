# Singbox 一键安装脚本

支持 VPS (Root) 和 Serv00/Hostuno 平台的 Singbox 多协议一键部署脚本。

## 核心功能
- **VPS 端**：支持 Vless-reality、Vmess-ws/Argo、Hysteria-2、Tuic-v5。
- **Serv00 端**：支持 Vless-reality、Vmess-ws(Argo)、Hysteria-2。
- **自动化**：一键安装、自动保活、节点信息导出。

## 安装命令

### 1. VPS 专用 (快捷方式: `sb`)
```bash
bash <(curl -Ls https://raw.githubusercontent.com/GamblerIX/singbox/main/sb.sh)
```

### 2. Serv00 / Hostuno 专用
```bash
bash <(curl -Ls https://raw.githubusercontent.com/GamblerIX/singbox/main/serv00.sh)
```
